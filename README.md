# DisateQ Vendor™

Plataforma de gestión comercial offline-first diseñada para pequeños negocios y emprendimientos que requieren una operación rápida, simple y resiliente.

---

# Filosofía del Proyecto

DisateQ Vendor™ prioriza:

- continuidad operacional,
- simplicidad real,
- resiliencia,
- velocidad,
- y compatibilidad con hardware básico.

La operación nunca debe detenerse.

# Principios Fundamentales

- Escalabilidad progresiva
