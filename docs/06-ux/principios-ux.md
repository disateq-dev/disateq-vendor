# UX Operacional

## Objetivo Principal

Velocidad operacional con mínima fricción.

---

# Principios UX

- Touch-first
- Keyboard-first
- Baja saturación visual
- Flujo corto
- Estados claros
- Operación rápida

---

# Filosofia

Diseñar pensando:

- rapidez operacional,
- continuidad,
- mínima fricción,
- y simplicidad real.

No diseñar solamente para verse bien.

---

# Prioridades UX

- vender rápido,
- cobrar rápido,
- continuar trabajando,
- evitar interrupciones,
- mantener foco operacional.
