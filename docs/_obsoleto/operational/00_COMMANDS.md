# DISATEQ Operational Commands

## Core Commands

### Vamos
Autorización explícita para avanzar.

---

### Hecho
Acción ejecutada correctamente.

---

### Confirmo
Validación correcta.

---

### Alto
Detener inmediatamente el flujo actual.

Usar cuando:
- exista deriva
- error conceptual
- autonomía excesiva
- pérdida de foco

---

### Resume
Solicitar resumen compacto operacional.

Formato esperado:
- estado actual
- problema
- siguiente paso

Sin narrativa extensa.

---

### Enfoque
Volver al objetivo inmediato.

Usar para cortar deriva contextual.

Ejemplo:

```text
Enfoque: CAJA
